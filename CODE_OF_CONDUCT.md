# 贡献者公约行为准则

## 我们的承诺

为了营造开放、友善的环境，我们作为贡献者和维护者承诺：无论年龄、体型、身体健全与否、民族、性别认同与表达、经验水平、国籍、个人外貌、种族、宗教信仰、性取向等，均以友善、尊重的态度对待社区每一位参与者。

## 我们的准则

有助于创造积极环境的行为包括但不限于：

* 使用友善、包容的语言
* 尊重不同的观点和经历
* 耐心接受建设性批评
* 关注对社区最有利的事情
* 友善对待其他社区成员

不可接受的行为包括但不限于：

* 使用与性有关的言语或图像，以及不受欢迎的性关注或挑逗
* 挑衅、侮辱或贬损的评论，以及人身攻击或政治攻击
* 公开或私下的骚扰
* 未经明确授权，公布他人的隐私信息，例如住址或电子邮箱
* 其他在专业环境中可能被认为不当的行为

## 我们的责任

项目维护者有责任诠释何谓“不可接受的行为”，并妥善、公正地纠正任何此类行为。

项目维护者有权利和义务删除、编辑或拒绝不符合本行为准则的评论、提交、代码、wiki 编辑、问题等贡献；如有必要，可暂时或永久地封禁任何贡献者。

## 适用范围

本行为准则适用于项目空间和公共空间。当个人代表项目或其社区时，本行为准则同样适用。代表项目或社区的情形包括但不限于：使用官方项目电子邮箱、通过官方社交媒体账号发言、作为指定代表参与在线或线下活动等。项目维护者将进一步定义和澄清如何代表项目及其社区。

## 执行

如发现滥用、骚扰或其他不可接受的行为，请通过 [联系邮箱] 向项目团队报告。所有投诉都将被及时、公正地审查和调查。项目团队有义务对报告者信息保密。

## 处罚

项目维护者若未切实遵守或执行本行为准则，可能会因项目领导层的决议而面临临时或永久性的处罚。
