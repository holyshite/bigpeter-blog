# 文章图片存储目录

此目录用于存储博客文章引用的图片文件。

## 使用规范

1. **文件命名**：使用小写字母、数字和连字符，如 `my-article-image.jpg`
2. **图片格式**：推荐使用 WebP、JPEG、PNG、SVG 等现代格式
3. **路径引用**：在 Markdown 文章中使用相对路径，如：
   ```markdown
   ![图片描述]({{ '/assets/images/your-image.jpg' | absolute_url }})
   ```
4. **组织方式**：可以按文章创建子目录，如 `2026/04/my-image.jpg`

## 最佳实践

- 压缩图片以减少页面加载时间
- 为重要图片添加 `alt` 描述
- 保持图片尺寸适中（宽度建议不超过 1200px）
- 使用响应式图片技术（如 `srcset`）
