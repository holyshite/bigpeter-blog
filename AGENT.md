# AGENT.md - AI助手使用指南

## 基本规则

1. **先读代码**：修改前必须理解现有代码
2. **保持风格**：遵循现有代码规范
3. **渐进修改**：一次只改一个问题
4. **测试验证**：修改后检查页面效果

## 可用操作

✅ **允许**
- 修复bug、优化性能
- 改进样式（不改变视觉效果）
- 添加新功能（需确认需求）
- 更新文档、注释
- 重构代码（保持功能不变）

❌ **禁止**
- 删除重要功能（未经确认）
- 大规模重写核心架构
- 修改部署配置（需明确授权）

## 沟通要点

- 提供明确指令
- 指出具体文件路径
- 说明期望结果
- 确认后再执行复杂操作

## 问题处理

遇到问题：
1. 检查控制台错误
2. 查看构建日志
3. 参考CLAUDE.md详细指南
